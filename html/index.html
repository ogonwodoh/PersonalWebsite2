Projects

FindPUBG

FindPUBG is the first website that allows users to find teammates for the game PlayersUnknown Battegrounds. I am working on the project with 3 other people. It is built with Django. More information can be found here.

SoundFlash

SoundFlash is an application that creates sound enabled flashcards. A user can add a song to the application and be quizzed on the name and composer of the song. This application was created to be uses in the class MUSIX 1001, An Introduction to Music at Barnard. More information can be found here.

Lazy Blogger

LazyBlogger is an automatic blogging utility that pulls in one's likes from Tumblr and YouTube, then displays those liked objects as a blogging utility. A user can also email the utility to creat a blog post. More information can be found here.
